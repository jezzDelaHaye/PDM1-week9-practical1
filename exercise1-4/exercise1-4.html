Instructions

You should only attempt this exercise if you have completed exercise 1-3. The goal of this exercise is to refactor the code from last practical's exercise 1-3 to work with output from your new, more flexible level generator. sketch.js contains the code from the sample solution for W8P2 > 1-3. Feel free to replace this with your own code.

Just like exercise 1-3, refactor the code so that a user will only need to make a few changes to the values of global constants in order to display a level generated by the level generator. An example saved level (scene.txt) is provided in the assets folder but you're encouraged to replace this with one you've created yourself.

As with 1-3, the key to this exercise is good use of global constants, arrays, and other advanced data structures (think object literals or classes). Avoid using hardcoded numbers / strings anywhere in your code other than setting the values of your global constants / variables or in the preload() function.

You will need to make modifications throughout your code.
